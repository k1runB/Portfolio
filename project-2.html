Project 2

Developed an image classifier with ResNeXt, blending innovation with precision. With grouped convolutions and smart transfer learning, this model tackles complex images efficiently, achieving impressive accuracy with refined elegance.

Code Link

Project Overview

This project aims to develop a robust deep learning-based classification model to differentiate between normal chest X-rays and cases of pneumonia, COVID-19, and tuberculosis. By combining the ResNeXt50 and VGG16 architectures, the model leverages advanced feature extraction techniques and fine-tuning for effective diagnosis of respiratory infections.

Key Components:
Dataset: The dataset consists of 7,135 X-ray images from Kaggle, organized into training, testing, and validation sets, with categories for Normal, Pneumonia, COVID-19, and Tuberculosis. Addressing significant data imbalance, particularly in COVID-19 and Tuberculosis classes, the project incorporates class balancing and data augmentation to enhance prediction accuracy.

Model Architectures:
ResNeXt50: Begins with convolution and max-pooling layers, progressing through grouped convolutions with ReLU activations, dropout, and L2 regularization. The model achieved 94.7% accuracy after 19 epochs.
VGG16: With 13 convolutional layers and 3 fully connected layers, VGG16 employs transfer learning with ImageNet-pretrained weights. By freezing convolutional layers and fine-tuning fully connected layers, the model reached 96.76% accuracy at 19 epochs.

This multi-level classification approach successfully tackles dataset imbalance and diagnostic challenges, providing healthcare with a fast, reliable tool for respiratory infection diagnosis.
